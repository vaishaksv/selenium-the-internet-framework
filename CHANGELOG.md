# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [2.0.0] - 2026-01-29

### Added
- Production-ready Selenium framework
- Page Object Model for 15 pages
- Auto-retry mechanism for failed tests
- Data-driven testing with TestNG DataProviders
- Soft assertions support
- Allure reporting integration
- Multi-environment support (dev/test/staging/prod)
- Cross-browser support (Chrome/Firefox/Edge/Safari)
- Parallel execution capabilities
- WebDriver event logging
- Test data builder pattern
- GitHub Actions CI/CD pipeline
- Comprehensive documentation

### Coverage
- Add/Remove Elements
- Checkboxes
- Dropdown
- JavaScript Alerts
- Drag and Drop
- Hovers
- Frames (iFrame/Nested)
- Multiple Windows
- Form Authentication
- Dynamic Loading
- Data Tables
- File Upload
- Key Presses
- Infinite Scroll
- Challenging DOM

## [1.0.0] - Initial Release

### Added
- Basic Selenium framework structure
